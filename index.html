November 2025 · Machine Learning

→ Customer Churn Prediction & Behavioral Segmentation

Designed an end-to-end churn prediction system by integrating 7 telecom datasets (7,000+ customers). Delivered ~85% ROC-AUC with the top 10% highest-risk customers accounting for ~77% churn, supporting targeted retention strategies. Evaluated 28 model configurations; selected CatBoost as the top performer using SMOTE and cross-validation.

Python CatBoost XGBoost Scikit-learn Pandas

October 2025 · Computer Vision

→ Semantic Segmentation for Urban Scene Understanding

Built a pixel-level semantic segmentation system for urban driving scenes, labeling roads, vehicles, and pedestrians. Achieved ~62% mIoU on an 11-class dataset and ~93% mIoU on binary road segmentation via transfer learning. Compared baseline models against PSPNet with pyramid pooling and dilated convolutions.

PyTorch OpenCV PSPNet Python NumPy

September 2025 · Unsupervised Learning

→ Fraud Detection via Unsupervised Pattern Discovery

Developed an unsupervised fraud detection workflow on 280,000+ financial transactions to identify anomalous behavior patterns without labeled data. Achieved well-separated behavioral clusters with up to 3× improvement over random baselines. Applied K-Means and Gaussian Mixture Models evaluated via Silhouette Score and Adjusted Rand Index.

Python K-Means GMM Scikit-learn Matplotlib
